## 七大必考題型

1. **數與運算**（25%）— 分數、小數四則運算、估算
2. **因數與倍數**（15%）— 質因數分解、GCD/LCM
3. **比率與百分率**（15%）— 比例、折扣、濃度
4. **代數**（15%）— 簡易方程式、未知數求解
5. **幾何**（15%）— 面積、體積、周長、角度
6. **速率**（10%）— 速率公式、相遇/追趕問題
7. **機統與規律**（5%）— 機率、統計圖表、數列規律

每種題型均提供**簡易 / 中等 / 難**三個級別的範例與詳解。
